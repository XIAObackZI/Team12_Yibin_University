# 个人博客项目

## 项目结构

- `index.html`: 项目主页，包含英雄区域、文章推荐、分类展示等。
- `articles.html`: 文章列表页面，支持分类浏览和筛选。
- `presentation.html`: 演示页面。
- `other_pages/`: 存放次要页面。
    - `team.html`: 团队介绍页面，包含搞怪风格设计和动画效果。
    - `publish.html`: 文章发布页面，支持表单提交和标签管理。
- `styles/`: 存放 CSS 样式文件。
    - `main.css`: 主要样式。
    - `components.css`: 组件样式。
    - `animations.css`: CSS 动画。
    - `dark-mode.css`: 暗黑模式样式。
    - `themes.css`: 主题样式。
- `scripts/`: 存放 JavaScript 脚本文件。
    - `page-transitions.js`: 处理页面之间的过渡动画。
    - `three-effects.js`: 实现团队页面的 Three.js 背景和 3D 模型效果。
    - `main.js`: 页面主要交互逻辑。
    - `animations.js`: JavaScript 动画逻辑。
    - `component-loader.js`: 加载通用组件。
    - `carousel.js`: 轮播图功能。
    - `content-generator.js`: 动态内容生成。
    - `pin-component.js`: 3D Pin 组件效果。
    - `music-player.js`: 音乐播放器功能。
    - `3d-card-component.js`: 3D 卡片效果。
    - `highlight.js`: 代码和文本高亮。
    - `utils.js`: 通用工具函数。
- `assets/`: 存放图片、声音等静态资源。
- `components/`: 存放可复用的 HTML 组件片段。
- `README.md`: 项目说明文件。
- `develop.md`: 开发相关文档。

## 功能模块

### 1. 页面过渡效果 (scripts/page-transitions.js)

实现了页面之间的平滑过渡动画效果，包括：

- **粒子背景效果**：多种类型粒子（圆形、方形、星形）随机分布并带有缓动动画
- **中心Logo旋转**：具有多层光环的旋转效果和内部闪光动画
- **进度条加载动画**：使用缓动函数实现自然加载效果
- **动态文字提示**：随机切换多种加载文案
- **多重动画结合**：包括缩放、旋转、闪烁和震动效果
- **完整动画序列**：从页面点击到完成加载的全流程动画处理

实现了无缝衔接的页面切换体验，增强用户体验的流畅度和视觉吸引力。

### 2. 团队展示页面 (other_pages/team.html)

展示团队成员信息的页面，包含：

- 搞怪风格的成员卡片和互动效果

- 有趣的动画效果（浮动、旋转、彩虹文字等）
- 派对模式功能
- 响应式设计
- 独特的表单互动效果

## JavaScript功能

团队页面包含以下主要JavaScript功能:

1. **triggerDemoAnimation()**: 
   - 触发演示动画效果
   - 激活页面覆盖层
   - 依次执行彩色碎屑、飞入图标、卡片动画和文字动画效果

2. **createConfetti()**: 
   - 创建彩色碎屑效果
   - 生成随机颜色、位置和大小的碎屑
   - 设置动画持续时间和自动移除

3. **createFlyingIcons()**: 
   - 创建飞入图标效果
   - 随机选择图标和颜色
   - 设置动画延迟和自动移除

4. **animateTeamCards()**: 
   - 为团队卡片添加入场动画
   - 设置动画延迟
   - 添加临时彩虹发光效果

5. **animateTexts()**: 
   - 为标题文本添加动画效果
   - 将文本拆分为单个字符并添加动画
   - 设置动画延迟

6. **派对模式功能**:
   - 点击派对按钮触发派对模式
   - 显示派对覆盖层
   - 生成五彩纸屑
   - 提供停止派对按钮

7. **页面加载和组件**:
   - 使用component-loader.js加载导航栏和页脚
   - 使用three-effects.js添加背景效果

# TechBlog网站项目

## 项目结构
```
TechBlog/
├── index.html             # 网站主页
├── articles.html          # 文章列表页面
├── team.html              # 团队页面(重定向)
├── other_pages/
│   └── team.html          # 团队页面(实际页面)
├── team_my.html           # 个人团队页
├── components/            # 组件目录
│   ├── sections/          # 页面区块组件
│   │   └── carousel-recommendations.html  # 推荐轮播组件
│   │
│   └── layout/            # 布局组件
│       ├── team-navbar.html  # 团队页导航栏组件
│       └── Navbar.html       # 主导航栏组件
└── scripts/               # 脚本文件
    ├── component-loader.js # 组件加载器
    └── three-effects.js    # Three.js效果脚本
```

## 组件功能说明

### 页面组件
- **index.html**: 网站首页，包含网站主要内容和功能入口
- **articles.html**: 文章列表页面，展示所有博客文章
- **team.html**: 重定向到other_pages/team.html的页面
- **other_pages/team.html**: 团队介绍页面，包含团队成员信息和多种动画效果
- **team_my.html**: 个人团队页面

### 布局组件
- **components/layout/team-navbar.html**: 团队页面专用导航栏
  - 包含网站LOGO、导航链接、搜索框、演示按钮、设置按钮、通知按钮和用户菜单
  - 响应式设计，适配移动端
  - 包含特殊的导航效果和动画

- **components/layout/Navbar.html**: 网站主导航栏
  - **结构组成**:
    - 左侧LOGO区: 包含TechBlog图标和文字标识
    - 中间导航链接区: 包含首页、文章、分类、开发团队、联系我们等主要导航项
    - 右侧功能区: 包含搜索框、通知按钮、用户菜单
    - 移动端汉堡菜单按钮: 在小屏幕设备上显示
  - **导航链接**:
    - 首页 (`/index.html`): 带有家图标
    - 文章 (`/articles.html`): 带有书本图标
    - 分类 (`/index.html#categories-section`): 带有标签图标，链接到首页的分类区块
    - 开发团队 (`/index.html#team-section`): 带有用户组图标，链接到首页的团队区块
    - 联系我们 (`/index.html#contact`): 带有信封图标，链接到联系区块
  - **搜索功能**:
    - 包含搜索输入框和搜索按钮
    - 响应式设计，在小屏幕上隐藏
  - **用户菜单**:
    - 用户头像按钮
    - 下拉菜单包含个人资料、设置和退出选项

### 内容组件
- **components/cards/hot-articles.html**: 热门文章轮播组件
  - **结构组成**:
    - 轮播容器: 包含多个文章卡片的滑动容器
    - 文章卡片: 每个卡片包含图片、标签、标题、摘要和作者信息
    - 控制按钮: 左右滑动的箭头按钮
    - 指示器: 底部的圆点指示当前位置
  - **功能特点**:
    - 自动轮播: 每5秒自动切换到下一组文章
    - 手动控制: 支持点击箭头按钮手动切换
    - 直接跳转: 支持点击底部指示器直接跳转到特定位置
    - 悬停暂停: 鼠标悬停在轮播区域时暂停自动轮播
    - 响应式适配: 自动适应不同屏幕尺寸，在窗口大小变化时重新加载
    - 循环滚动: 到达末尾后平滑回到开始位置
    - 动画效果: 平滑的过渡动画和悬停效果
  - **关键JavaScript函数**:
    - `moveCarousel(direction)`: 控制轮播向前或向后移动
    - `goToSlide(index)`: 跳转到指定索引的幻灯片
    - `updateIndicators()`: 更新底部指示器的状态
    - `startAutoplay()`: 开始自动轮播
    - `stopAutoplay()`: 停止自动轮播
  - **轮播控制逻辑**:
    - 使用CSS transform属性平滑移动卡片容器
    - 循环逻辑确保无限滚动效果
    - 防抖动处理确保动画平滑
    - 窗口大小变化时自动重新计算尺寸
