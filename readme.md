# 个人博客项目

## 项目结构

- `index.html`: 项目主页，包含英雄区域、文章推荐、分类展示等。
- `articles.html`: 文章列表页面，支持分类浏览和筛选。
- ~~`article-detail.html`~~: ~~文章详情页面，支持点赞、收藏功能~~。(已移除，使用assets/Article_detail_page/下的详情页模板)
- `presentation.html`: 演示页面。
- `other_pages/`: 存放次要页面。
    - `team.html`: 团队介绍页面，包含搞怪风格设计和动画效果。
    - `publish.html`: 文章发布页面，支持表单提交和标签管理。
- `styles/`: 存放 CSS 样式文件。
    - `main.css`: 主要样式。
    - `components.css`: 组件样式。
    - `animations.css`: CSS 动画。
    - `dark-mode.css`: 暗黑模式样式。
    - `themes.css`: 主题样式。
- `scripts/`: 存放 JavaScript 脚本文件。
    - `page-transitions.js`: 处理页面之间的过渡动画。
    - `three-effects.js`: 实现团队页面的 Three.js 背景和 3D 模型效果。
    - `main.js`: 页面主要交互逻辑。
    - `animations.js`: JavaScript 动画逻辑。
    - `component-loader.js`: 加载通用组件。
    - `carousel.js`: 轮播图功能。
    - `content-generator.js`: 动态内容生成。
    - `pin-component.js`: 3D Pin 组件效果。
    - `music-player.js`: 音乐播放器功能。
    - `3d-card-component.js`: 3D 卡片效果。
    - `highlight.js`: 代码和文本高亮。
    - `utils.js`: 通用工具函数和本地存储功能。
- `assets/`: 存放图片、声音等静态资源。
- `components/`: 存放可复用的 HTML 组件片段。
- `README.md`: 项目说明文件。
- `develop.md`: 开发相关文档。

## JavaScript功能

团队页面包含以下主要JavaScript功能:

1. **triggerDemoAnimation()**: 
   - 触发演示动画效果
   - 激活页面覆盖层
   - 依次执行彩色碎屑、飞入图标、卡片动画和文字动画效果
   - 添加最终过渡动画效果（圆形展开效果、火箭飞行动画和文字提示）
   - 动画结束后自动跳转到演示页面(presentation.html)

2. **createConfetti()**: 
   - 创建彩色碎屑效果
   - 生成随机颜色、位置和大小的碎屑
   - 设置动画持续时间和自动移除

3. **createFlyingIcons()**: 
   - 创建飞入图标效果
   - 随机选择图标和颜色
   - 设置动画延迟和自动移除

4. **animateTeamCards()**: 
   - 为团队卡片添加入场动画
   - 设置动画延迟
   - 添加临时彩虹发光效果

5. **animateTexts()**: 
   - 为标题文本添加动画效果
   - 将文本拆分为单个字符并添加动画
   - 设置动画延迟

6. **createFinalAnimation()**: 
   - 创建中央圆形展开效果

    - 平滑循环逻辑: 优化在轮播到开始和结束位置时的平滑过渡效果

  - **核心函数**:
    - `getCardWidth()`: 动态计算卡片宽度
    - `getMaxPosition()`: 计算轮播最大位移值

    - `goToSlide(index)`: 跳转到指定索引的幻灯片
    - `updateIndicators()`: 更新底部指示器的状态
    - `updateCardWidths()`: 确保所有卡片宽度一致
    - `initCarousel()`: 初始化轮播组件

    - 左右箭头按钮和底部指示器导航
